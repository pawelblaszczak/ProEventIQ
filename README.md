# ProEventIQ

Professional Event Management System

## Project Structure

- **api/** - API specifications (OpenAPI)
- **backend/** - Spring Boot backend application
- **frontend/** - Angular frontend application
- **doc/** - Project documentation

## Getting Started

### Backend

```bash
cd backend
./mvnw spring-boot:run
```

### Frontend

```bash
cd frontend
npm install
npm start
```
